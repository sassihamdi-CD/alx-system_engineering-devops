# 0x14 MySQL

## Description
This directory contains tasks related to configuring and managing MySQL databases as part of the ALX Africa program. These tasks include setting up MySQL servers in a primary-replica configuration, creating databases and users, and automating database backups.

- 1-create_user.sql
- 2-replica_configuration
- 3-backup_script

## 0-create_db.sql

- **File:** `0-create_db.sql`

## 1-create_user.sql

## 2-replica_configuration

- **File:** `2-replica_configuration`

## 3-backup_script
This script generates a compressed tar.gz archive from a MySQL dump, containing all databases on the server.

- **File:** `3-backup_script`
